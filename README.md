CList
=====

Linked list data structure in pure C implemented for one cute girl.

###Supported command:

* Create list
* Add item (or items)
* Get length
* Get head / tail
* Replace all occurrences of n to m
* Two list concatenation
* List inversion
* Sorting
* StartsWith and isSublist methods

###Also some GUI =))
* Read list from console
* Write to console
